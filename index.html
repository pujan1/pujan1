Pi Cloud

PaaS service that can add or remove the nodes in real-time without restarting the server.

• Pi Cloud is a Raspberry Pi based PaaS service that can add or remove the nodes in real-time without restarting the server. It is capable of downloading an app, its dependencies and installing on the node of the choice.
• It consists of 3 or more Raspberry pi with one being the controller and other being the nodes. This idea can be implemented as a functioning distributed system.
• Controller accepts the requests from the user and routes it to the appropriate node. It registers the new nodes being added to the hub which can be then used to install and run apps. The controller will automatically install all the dependencies it needs to run the application on the appropriate node. Node as basically the models which stores and runs the app as per directed by the controller. a client or a user doesn't have to interact with node anytime.
• The Technicality:
The base controller is installed with a Python Fabric, which is responsible for accessing the nodes from the main controller. it will take the control to the node and start using the application or if it is not installed then it will install it and then start it.
• The Nginix is the router which is used for routing the requests to the appropriate nodes. A shell script is written which helps fabric by getting the name and the dependencies of the application.
